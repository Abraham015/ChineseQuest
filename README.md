# ChineseQuest

Aplicacion React para estudiar chino mandarin.

## Desarrollo

```bash
npm install
npm run dev
```

Servidor local: `http://127.0.0.1:5174/`

## Build

```bash
npm run build
```

La salida se genera en `dist`.

## Despliegue en Netlify

Configuracion incluida en `netlify.toml`:

- Build command: `npm run build`
- Publish directory: `dist`

En Netlify puedes conectar este repositorio y dejar que detecte el archivo `netlify.toml`.
